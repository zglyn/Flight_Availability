# Flight_Availability

## Features 
Users can find available flights from a origin airport to an airport at their destination between a range of dates. On those shown flights, they can check the number of available seats. 

## How to Use 
After downloading the files, run py init_db.py to generate an updates flights.db file. 
Then, run py app.py to run the website. 
Finally, go to http://127.0.0.1:5000
